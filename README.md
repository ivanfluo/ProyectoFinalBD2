# ProyectoFinalBD2

Proyecto final del curso Bases de Datos 2.

## Tecnologías utilizadas

- Oracle Database
- SQL Server
- PostgreSQL
- SQL Developer
- SSIS
- SSRS
- Visual Studio

## Objetivo

Desarrollar una solución de Business Intelligence capaz de integrar, transformar y analizar información académica proveniente de múltiples fuentes de datos.

## Arquitectura

- Base transaccional Oracle
- Base transaccional SQL Server
- Base PostgreSQL
- Archivos Excel y TXT
- Área Staging
- Data Warehouse
- Procesos ETL
- Reportería y análisis
